X-Builders Keyboard and Mouse Reference

MOUSE ACTIONS

Mouse Button Corresponding Action
Left Mouse Button Create block.
Right Mouse Button Delete block.
Center Mouse Button Pick block
Mouse Scroll Select block from block panel

Player movement

Key Corresponding Action
WASD or Arrow Keys Move Player
F Fly up
Ctrl Fly down
Shift Run
Space Bar Jump / Toggle Fly / Climb Up Ladders

KEY ACTIONS

Key Corresponding Action
Esc Toggle Game Menu
E Toggle Block Menu
- Simulate destroy click
+ Simulate create click
0 Simulate middle-mouse click
< Select left block from menu
> Select right block from menu
Tab Select in midair
Alt (Hold) open block tool pallete

OTHER ACTIONS

Key Corresponding Action
F3 Toggle dev stats
F4 Toggle UI visibility
F5 Toggle wireframe
F6 Toggle bounding box mode

BLOCK TOOLS

Hold ALT to switch between all block tools. Below are the most common block tools:

Activation Key Corresponding Action
1 Default mode.
2 Boundary of blocks.
(K to place boundary inside block)
3 Plane of blocks
4 Line of blocks.
(Scroll to change length)
Ctrl+C Copies a set of blocks within a boundary.
(K to place boundary inside block)
Ctrl+V Pastes a copied boundary of blocks.
(Scroll to change vertex, R to rotate)
Additionally, you can use the arrow keys and shift+up/down arrow key to move the paste point after it has been set
