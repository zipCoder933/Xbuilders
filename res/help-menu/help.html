X-Builders Keyboard and Mouse Reference

MOUSE ACTIONS

Mouse Button Corresponding Action
Left Mouse Button Create block. (Can be changed with "swapMouseButtons")
Right Mouse Button Delete block. (Can be changed with "swapMouseButtons")
Center Mouse Button Picks selected block from world
Mouse Scroll Select block from horizontal block panel

KEY ACTIONS

Key Corresponding Action
Ctrl+Shift+P Toggle dev stats
Shift+T Toggle terrain updater
Q Drop item from my item list
Esc, M Toggle Game Menu
I Toggle Full Block Menu
WASD or Arrow Keys, Space Bar Move Player
J Toggle auto-forward
Shift Run (Hold down while moving)
Space Bar Climb Up Ladders
Shift Climb Down Ladders
P Toggle pass-through mode
F Fly up
Shift+F (When flying): Fly down
- Simulate destroy click
+ Simulate create click
0 Simulate middle-mouse click
< Select left block from menu
> Select right block from menu
Tab Toggle ray, hit air mode
Ctrl + Scroll Change ray air mode distance
Ctrl + Shift + E Enable/Disable Entity rendering and updating
Shift+X Cancel Sunlight Propagation

BLOCK MODES

ACITONS

Key Corresponding Action
Shift + Scroll Change block mode size
R Rotate block paste boundary
V Change anchor vertex of block paste boundary

MODES

Key Alternative Key Corresponding Action Alternative Action
(to toggle between alternative actions, press the hotkey again)
1 default mode, and only places a single block.
2 Ctrl+L Places a line of blocks. Erases a line of blocks.
3 Ctrl+R Repaints all of a specific selected block type within a specific radius. Erases all of a specific selected block type within a specific radius.
4 Ctrl+P places a 2D plane of blocks.
5 Ctrl+K Sets a regtangular boundary of blocks Erases a regtangular boundary of blocks
6 Ctrl+S Sets a sphere within the boundary Sets a hollow sphere within the boundary
Ctrl+C Copies a set of blocks within a boundary.
Ctrl+V Pastes a copied boundary of blocks.
Ctrl+Shift+V Pastes a copied boundary of blocks, but does not delete blocks that are already there.

GAME SETTINGS

CHUNK DISTANCE

The chunk distance option in the game's "Settings" menu controls the maximum chunk view distance. The chunks have a width of 16. So your chunk distance value should be some multiple of 16.
(160 is the default chunk distance value.)
